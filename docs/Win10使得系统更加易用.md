## 快捷键

### Calibre

1. 重新启动 Ctrl-r
2. 快速选中虚拟书库 Ctrl-t
3. 切换到上一个书库 Ctrl-Alt-P
4. 跳转到下一个匹配项目 N

5. 添加书籍 A、移除书籍Del、保存到磁盘 S
6. 阅读 V、阅读特定格式 Alt-v
7. 编辑元数据 E、编辑书籍 T
8. 浏览批注 B、转换书籍 C、编辑目录 K



### MPV

- 播放、暂停、切换文件
